# 客户端对象

可分为[实例和控制台](../../client)

| 名称        | 描述             | 值类型               |
| ----------- | ---------------- | -------------------- |
| custom_name | 自定义名称       | string/null          |
| address     | IP:端口          | string               |
| guid        | 客户端唯一识别码 | string               |

## Instance

| 名称        | 描述             | 值类型               |
| ----------- | ---------------- | -------------------- |
| full_info   | 实例完整信息     | [FullInfo](fullInfo) |

## Console

| 名称               | 描述             | 值类型      |
| ------------------ | ---------------- | ----------- |
| subscribing_target | 当前订阅的对象   | string      |
