# 断开连接

Host主动断开前发送，用于说明断开原因

- 网页控制台
  - [ ] 收
- 实例
  - [x] 收
  - [ ] 发

## type

`event`

## subType

`disconnection`

## data

| 名称   | 描述       | 值类型 |
| ------ | ---------- | ------ |
| reason | 断开的原因 | string |

- `发送的数据包存在问题：...` 数据包不合法
- `验证失败，请稍后重试` 验证未通过
- `你还未通过验证` 还未通过验证时先发送了其他数据包
- `验证超时` ~~太慢了我等到花都谢了~~
