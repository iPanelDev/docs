# 成员

所有数据包均应包含以下成员

| 名称     | 描述           | 值类型                   | 可能的值                                     |
| -------- | -------------- | ------------------------ | -------------------------------------------- |
| type     | 数据包的主类型 | string                   | `action` `event`                             |
| sub_type | 数据包的子类型 | string                   | [`verify_request`](action/verifyRequest) ... |
| data*    | 数据部分       | 取决于`type`和`sub_type` |                                              |

:::note
除此之外，[这里还有一些附加成员](additionalMembers)
:::

## type

决定了数据包的主要目的

- `action`
  - 请求数据
  - 执行服务器相关操作
- `event`
  - 服务器状态变更
  - 验证结果通知
  - 断开原因提示

## sub_type

进一步补充数据包的类型内容

## data

数据内容

*为null时将不会出现在数据包中；发送数据包时缺少此值则默认为null
