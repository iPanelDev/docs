## 创建用户

### 使用非交互模式

```txt
user create example 114514 1
13:27:32 Info  创建成功
```

### 使用交互模式

```txt
user create

── 用户名 ────────────────────────────────────────────────────
▪ 长度应大于3
▪ 不含有空格（ ）
▪ 不含有特殊字符（"'\@）
▪ 不与已有的用户名重复

> example

── 输入密码 ──────────────────────────────────────────────────
▪ 长度应大于6
▪ 不含有空格（ ）
▪ 不含有敏感字符（"'\）
▪ 不建议于其他密码相同（如服务器连接密码、QQ或微信密码）
▪ 推荐大小写字母数字结合

> ******

── 用户权限 ──────────────────────────────────────────────────
▪ 使用键盘的 <↑> 和 <↓> 进行选择
▪ 使用键盘的 <Enter> 或 <Space> 进行确认

管理员: 允许控制服务器、新建修改删除用户

── 描述（可选） ──────────────────────────────────────────────
▪ 你可以使用键盘的 <Enter> 选择跳过


13:21:36 Info  创建成功
╭──────────┬─────────╮
│  用户名  │ example │
├──────────┼─────────┤
│ 权限等级 │ 管理员  │
│   密码   │ ******  │
│   描述   │         │
╰──────────┴─────────╯
```

## 修改用户

### 使用交互模式

```txt
user edit

── 选择要编辑的用户 ───────────────────────────────────────────
▪ 使用键盘的 <↑> 和 <↓> 进行选择
▪ 使用键盘的 <Enter> 或 <Space> 进行确认

example (管理员)

── 输入密码 ──────────────────────────────────────────────────
▪ 长度应大于6
▪ 不含有空格（ ）
▪ 不含有敏感字符（"'\）
▪ 不建议于其他密码相同（如服务器连接密码、QQ或微信密码）
▪ 推荐大小写字母数字结合

> ******

── 用户权限 ──────────────────────────────────────────────────
▪ 使用键盘的 <↑> 和 <↓> 进行选择
▪ 使用键盘的 <Enter> 或 <Space> 进行确认

助手:   允许控制服务器

── 描述（可选） ──────────────────────────────────────────────
▪ 你可以使用键盘的 <Enter> 选择跳过


13:23:50 Info  修改成功

```

## 删除用户

### 使用交互模式

```txt
user remove

── 选择要删除的用户 ──────────────────────────────────────────
▪ 使用键盘的 <↑> 和 <↓> 进行选择
▪ 使用键盘的 <Enter> 或 <Space> 进行确认

example (管理员)
13:25:03 Info  删除成功
```
